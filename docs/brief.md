# ManaMind Website Design Brief

## 1. Project Overview

ManaMind is building the core intelligence that powers fully autonomous AI testing for video games, with long-term ambitions to become the default operating system for robotics across all segments.

The current website was created quickly and doesn't reflect our maturity, capability, or ambition. We need a premium, future-proof site that communicates our product, vision, credibility, and momentum to multiple audiences.

### Primary goals of the new site:
- Convert game studios into inbound leads and pilot requests.
- Establish authority in AI, robotics, and autonomy.
- Present ManaMind as a high-end deep-tech company, not a scrappy prototype.
- Build trust with investors & partners.
- Attract top-tier talent.

### Primary audiences:
- QA Directors, CTOs, and Production Leads at game studios.
- Investors
- Talent AI/ML engineers.

## 2. Brand, Positioning & Personality

### Brand Essence
- Visionary/Startup-y
- Fun but not childish
- High-performance
- Technical
- Clean
- Sci-fi without being gimmicky
- Confident, not loud

### Tone of Voice
- Smart, concise, direct.
- Ambitious but also fun.
- Technically credible without jargon overload.
- Premium deep-tech energy.
- Language that feels purposeful and modern.

### Key Messaging Pillars
- "Autonomous QA. Zero-shot. No code access."
- "We figured out a way to marry the intelligence of humans with the speed of machines, delivering fully autonomous testing that's fast, adaptable, and infinitely scalable"
- "The tri-bot system: Probe, Overseer, Scribe."
- "Spinoff bots - a smaller version of our main system that automates a specific type of testing"
- "From gaming to robotics: building the default OS for robots."
- "We are building the most fun company to work at!"

## 3. Functional Requirements

### Base pages:
- **Homepage** - hero animation/interaction, key value, social proof
- **Product** - Hivemind info (proprietary model), Command Centre (app), Bots (agents), capabilities, workflows, demo content
- **How It Works** - zero-shot approach, autonomy, tech overview
- **Case Studies** - option to easily publish new ones; should support text, images, and videos
- **About** - mission, team, vision, investors, story
- **Careers** - paramount to make it amazing to attract talent
- **Contact** - simple, low friction, high-converting
- **Blog/News** - to help with SEO and store PR features

### Technical Requirements
- Fast load times, globally optimised.
- Fully responsive on all devices.
- Easy for team to update pages after project handover with clear documentation and guidance.
- SEO optimised

### Integrations:
- Contact form > email, & CRM
- Google Analytics
- Calendar integration for demo calls

## 4. Visual Direction

### General Aesthetic
- High-end sci-fi but also fun.
- Structured, geometric, computational feel.
- Clean typography and lots of breathing room.
- Zero "indie game" or "cute AI" vibes.
- No stock photos or generic illustrations.

### Motion & Interactions
- Subtle micro-interactions
- Smooth transitions (fade/slide)
- One strong hero animation that communicates "autonomous intelligence in motion"
- Diagram-level interactions (tri-bot explanation)

### Assets to Use
- ManaMind logos
- Product UI mockups
- Screenshots and demo clips of the bots
- Progression & Replicator Bots
- ProgressionBot Game 2
- Wayfinder
- Company/Team photos

## 6. Must-Have Deliverables
- Complete, production-ready website
- Design system (colours, typography, spacing, button styles, components)
- Animation specifications
- Source files
- Quick handover guide or Loom walkthrough

## 7. Must-Have vs Nice-to-Have

### Must-Have
- Strong sci-fi premium aesthetic
- Clear product demo
- Smooth animations
- High performance
- Modular design
- Easy editing for non-technical team members

### Nice-to-Have
- 3D elements
- Interactive diagrams
- Dark/light mode toggle
- Custom iconography

## 8. What to Avoid
- Stock photos
- Overly cute / playful visuals
- Overly academic technical descriptions
- Messy font choices
- Animation overload causing slow load times
