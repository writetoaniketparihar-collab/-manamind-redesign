# Design Research — Reference Websites

## Nunu (nunu.ai) — Competitor
**Emil likes:** "How it works" section

- "How it Works" uses 3 numbered steps, each with visual + description — simple, clear workflow
- Dark purple background (#0A0A1C), magenta/purple accents
- Card-based feature layouts, curved section dividers
- Poppins font, bold hero headlines (4rem desktop)
- Smooth hover transitions, button shadow effects
- Floating geometric shapes for visual interest

## Included Games (included.games) — Partner
**Emil likes:** Mouse animation at the start of the home page

- Particle.js animations with hover-triggered "grab" interactions (purple particles #9069ff connecting on mouse proximity)
- GSAP ScrollTrigger for horizontal scrolling team carousel
- Inter font (weights 300–800), vibrant purple palette
- Vertical scroll layout with horizontal scroll sub-sections
- Discord CTA prominent in hero

## Slite (slite.com)
**Emil likes:** Cleanliness and the strikeout text (very bold)

- Extreme cleanliness — generous whitespace (100px vertical section padding, 40px horizontal)
- Max-width containers (1280px) with centered content
- **Strikeout text effect** on hero headline — bold typographic statement
- Infinite carousel with hover-pause for logos/social proof
- Subtle card lift on hover (translateY -2px)
- Gradient text treatments and gradient CTA buttons
- CSS variables for theming, smooth 0.3s transitions
- Clip-path masks for sophisticated reveal effects

## Lexi (lexi.tech)
**Emil likes:** Cleanliness — but finds it "a bit too generic"

- Clean, professional layout with Public Sans font
- Dark blue (#001326) header, teal (#75c6e7) accents
- Full-width hero with gradient overlay
- Card hover scale (1.05) with image blur effects
- Logo carousel with skewed geometric backgrounds
- **Takeaway:** Avoid the overly corporate/safe look — ManaMind needs more personality

## Momentic (momentic.ai)
**Emil likes:** Simple, clean design with a very clear message

- Clear single-line value prop hero: "Ship fast with AI testing"
- Dark/light contrast sections alternating
- GSAP-powered logo marquee scroll (pauses when off-screen)
- ASCII generative cross-pattern background animation (sine/cosine waves)
- Customer logo grid with hover opacity reveals
- Inter font, minimal color palette (dark neutrals + blue #4d65ff accent)
- Testimonial blocks: quote cards + company logos + speaker roles
- Business email validation on demo form

## Design Direction for ManaMind

### Hero Section
- Particle/generative animation adapted to ManaMind's palette (#2E1339 bg, #00FF96 neon green, #FF4C54 red)
- Inspired by Included Games' particle interactions + Janus' hero visualization concept
- Strikeout text a la Slite for bold messaging (e.g., ~~Manual QA~~ → Autonomous AI Testing)
- Clear single-line value prop like Momentic

### How It Works / Product
- Nunu-style numbered steps for the tri-bot system (Probe, Overseer, Scribe)
- Card-based feature breakdowns

### Social Proof
- GSAP ScrollTrigger horizontal scrolling logo carousel (investors + partners)
- Testimonial cards like Momentic

### General
- Inter or similar clean sans-serif font
- Generous whitespace like Slite (100px section padding)
- Dark mode default with #2E1339 background
- Neon green (#00FF96) primary, red (#FF4C54) highlight accents
- Subtle micro-interactions: card lifts, hover opacity, smooth 0.3s transitions
- Avoid: stock photos, generic corporate feel (Lexi trap), animation overload
